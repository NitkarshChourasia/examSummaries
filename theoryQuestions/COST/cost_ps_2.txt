Question: Explain me the different applications of the following Central Tendency with an example:
Mean - Arithmetic, Geometric, Harmonic
Median
Mode
Quartiles
Deciles
Percentiles

Solution: 

Mean:
Arithmetic Mean: The arithmetic mean is the sum of a set of numbers divided by the number of values in the set. For example, the arithmetic mean of the numbers 2, 3, 4, and 5 is (2 + 3 + 4 + 5) / 4 = 3.5.

Geometric Mean: The geometric mean is the nth root of the product of n numbers. For example, the geometric mean of 2, 3, 4, and 5 is (2 * 3 * 4 * 5)^(1/4) = 3.6.

Harmonic Mean: The harmonic mean is the reciprocal of the arithmetic mean of the reciprocals of a set of numbers. For example, the harmonic mean of 2, 3, 4, and 5 is 1 / ((1/2 + 1/3 + 1/4 + 1/5) / 4) = 2.6.

Median: The median is the middle value in a set of numbers when they are arranged in order from least to greatest. For example, the median of 2, 3, 4, and 5 is 4.

Mode: The mode is the most frequently occurring value in a set of numbers. For example, the mode of 2, 3, 4, and 5 is 5.

Quartiles: Quartiles are values that divide a set of numbers into four equal parts. The first quartile (Q1) is the median of the lower half of the data set; the second quartile (Q2) is the median; and the third quartile (Q3) is the median of the upper half of the data set. For example, if we have the numbers 1, 2, 3, 4, 5, 6, 7 then Q1 = 2; Q2 = 4; and Q3 = 6.

Deciles: Deciles are values that divide a set of numbers into ten equal parts. The first decile (D1) is the 10th percentile; the second decile (D2) is the 20th percentile; and so on up to D10 which is the 100th percentile. For example, if we have the numbers 1 to 10 then D1 = 1; D2 = 2; D3 = 3; D4 = 4; D5 = 5; D6 = 6; D7 = 7; D8 = 8; D9 = 9; and D10 = 10.

Percentiles: Percentiles are values that divide a set of numbers into 100 equal parts. The first percentile (P1) is the 1st percentile; the second percentile (P2) is the 2nd percentile; and so on up to P100 which is the 100th percentile. For example, if we have the numbers 1 to 100 then P1 = 1; P2 = 2; P3 = 3; P4 = 4; P5 = 5; P6 = 6; P7 = 7; P8 = 8; P9 = 9; and so on up to P100 which is 100.


Question: Explain me the different applications of the following measures of dispersion with an example:
Range
Variance
Standard Deviation
Coefficient of Variation

Solution: 

Range: Range is the difference between the highest and lowest values in a set of data. For example, if a set of data contains the values 2, 4, 6, 8, 10, the range is 10 - 2 = 8.

Variance: Variance is a measure of how spread out a set of data is. It is calculated by taking the average of the squared differences from the mean. For example, if a set of data contains the values 2, 4, 6, 8, 10, the variance is (2-6)^2 + (4-6)^2 + (6-6)^2 + (8-6)^2 + (10-6)^2 = 16.

Standard Deviation: Standard deviation is a measure of how spread out a set of data is. It is calculated by taking the square root of the variance. For example, if a set of data contains the values 2, 4, 6, 8, 10, the standard deviation is sqrt(16) = 4.

Coefficient of Variation: The coefficient of variation is a measure of relative dispersion. It is calculated by dividing the standard deviation by the mean. For example, if a set of data contains the values 2, 4, 6, 8, 10, the coefficient of variation is 4/6 = 0.67.

Question: Explain me the different purposes of the following sampling methods with an example:
Stratified Sampling
Systematic Sampling
Cluster Sampling

Solution: 

Stratified Sampling: Stratified sampling is a method of sampling that involves dividing a population into smaller groups (strata) based on certain characteristics, and then randomly selecting members from each group. For example, if you wanted to survey people about their political views, you could divide the population into strata based on age, gender, and location, and then randomly select people from each stratum to participate in the survey.

Systematic Sampling: Systematic sampling is a method of sampling that involves selecting members of a population at regular intervals. For example, if you wanted to survey people about their favorite type of music, you could select every 10th person from a list of people to participate in the survey.

Cluster Sampling: Cluster sampling is a method of sampling that involves dividing a population into smaller groups (clusters) and then randomly selecting one or more clusters to participate in the survey. For example, if you wanted to survey people about their opinions on a new product, you could divide the population into clusters based on geographic location and then randomly select one or more clusters to participate in the survey.

Point Estimation
Interval Estimation

For example, if you wanted to estimate the average height of all adults in the United States, you could take a sample of adults and calculate the average height of that sample. That average would be your point estimate of the population parameter.

Interval Estimation: Interval estimation is a method of statistical inference that uses sample data to calculate an interval (or range) of values which is likely to contain the unknown population parameter. For example, if you wanted to estimate the average height of all adults in the United States, you could take a sample of adults and calculate the average height of that sample, as well as the standard deviation. You could then use this information to calculate a confidence interval, which would be your interval estimate of the population parameter.

Chi-Square = (Observed Frequency - Expected Frequency)2 / Expected Frequency

Chi-Square = (60 - 50)2 / 50 + (40 - 50)2 / 50 = 10 / 50 + (-10)2 / 50 = 10 + 100 / 50 = 110 / 50 = 2.2

The researcher can then compare this value to a critical value from a Chi-Square table to determine if there is a significant difference between the expected and observed frequencies.
