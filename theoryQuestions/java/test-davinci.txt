NOTE: I am using mac os, but I prefer a solution that will work on all platforms.
My current solution:
<code>sed -E ':a;N;$!ba;s/\n\n/\n/g' $1
</code>
Any ideas?


A:

If you are using the GNU version of <code>sed</code>, then the following will work:
<code>sed -zr 's/\n\n+/\n/g' filename
</code>
This command should work in Linux, Cygwin, and MacOS.
However, this does not work in FreeBSD because the <code>-z</code> option is not supported there.
